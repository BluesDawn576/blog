Blender 学习笔记 — 快捷键

发表于2023-03-11|更新于2023-03-29|笔记blender
|阅读量:

Blender 3.4.x

[.] 可选参数

基本操作

鼠标中键 视角旋转
Shift + 鼠标中键 视角位移

Shift + A 添加对象
Shift + C 游标返回原点
Shift + D + [X/Y/Z] 复制物体
Shift + 1 物体显示/隐藏

A 全选
B 框选
C 刷选
X 删除
Shift 多选
~ 快捷轮盘

物体变换

G + [X/Y/Z] + [位移距离] 移动
R + [X/Y/Z] + [角度] 旋转
S + [X/Y/Z] + [缩放大小] 缩放

编辑模式

Tab 进入/退出编辑模式

1 点选择模式
2 线选择模式
3 面选择模式

E + [X/Y/Z] 挤出选区
F 填充面
I 内插面
Ctrl + B + [鼠标滚轮] 边线倒角
Ctrl + Shift + B + [鼠标滚轮] 顶点倒角
Ctrl + R + [鼠标滚轮] 环切
GG 滑移

Alt + Z 透视模式
Alt + S 法向缩放

Alt + 鼠标左键 快速选择

P 分离实体

视角变换

[Ctrl] + 小键盘1 前视图（Ctrl反向）
小键盘3 右视图
[Ctrl] + 小键盘7 顶视图（Ctrl反向）
小键盘9 左视图

小键盘5 切换透视/正交

小键盘0 摄像机视角
Ctrl + Alt + 小键盘0 当前视角设为摄像机视角
