为你的Minecraft安装Replay Mod，实现录制功能

发表于2020-04-08|更新于2020-04-08|好东西
|阅读量:

什么是ReplayMod

ReplayMod是由 CrushedPixel 和 Johni0702 研发的一款能够录制、回放和导出视频的模组，体积小但功能非常强大。生成的回放文件非常小，平均一小时不到10MB。然后你还可以通过模组自带的回放中心去观看和分享录像。并且有个好处是它还可以跟光影搭配

包含的功能：游戏录制与回放、剪辑回放内容、制作摄像机路径动画、导出视频、回放中心等

缺点：导出的视频无声音

1.准备工作

下载前请先确定你的Minecraft版本

对于1.12.2及以下的版本，需要安装forge加载器。

1.14.4及以上的版本，安装fabric加载器。

forge安装教程请寻找站外其他文章，这里我们来讲一下fabric的安装方法

2.fabric加载器安装

首先我们访问fabric官网，找到加载器安装程序，这里我们下载exe格式的安装器

打开安装器，如果你用的是非官方启动器，安装位置选择.minecraft实际路径，我的推荐是创建一个新的启动器配置

然后就是耐心等待（）

安装完毕后，目录就会多出一个版本

当然也可以使用第三方启动器一键安装

例(HMCL)：

3.需要的前置mod

到这里fabric并没有完成，你还需要安装fabric api

找到我们要安装的版本，比如我是1.14.4，就去找对应版本。扔到mods文件夹

为了Optifine可以运行在fabric，你还需要安装OptiFabric这个前置mod

还有一点比较坑的是，fabric不自带mod列表功能，因此还需要Mod Menu

4.安装ReplayMod（Forge和Fabric通用）

安装本体

前面fabric加载器的工作已经基本完成，现在来安装ReplayMod（依然以1.14.4为例）

下载链接：https://www.replaymod.com/download/

下载后扔进mods文件夹

安装FFmpeg解码器（渲染视频会用到）

此处仅适用于Windows，其他系统请看官方文档

下载链接：https://ffmpeg.zeranoe.com/builds/

版本默认，根据你的电脑位数选择32位或64位。下载后解压放进.minecraft根目录的ffmpeg文件夹（自行创建），结构如下图所示

另外你还需要批处理文件来安装FFmpeg（https://replaymod.com/files/ffmpeg-path-installer.bat），放进ffmpeg文件夹，双击运行，出现成功字样即可

到这里安装流程基本结束，现在我们可以进游戏了

早期bug，新版本已修复

官方文档：https://www.replaymod.com/docs/
