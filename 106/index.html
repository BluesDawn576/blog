愿能在更开放的平行世界相遇

酷Q是一款大家熟知的QQ机器人，由于本体是基于windows系统研发的，所以Linux/MacOS用户不能直接运行。好在官方发布了Docker镜像， 并且也基于wine技术，因此我们可以利用Docker和wine来运行酷Q。

酷Q on Docker 基于的是 Docker 和 wine 技术 ，由于wine相对于windows仍然有差距，因此可能会出现部分应用不兼容的情况。
